GET /api/pokemon

Methods: GET, HEAD

Query Params (optional):

  • name (string) name contains
  • type (string) exact match against type array
  • weakness (string) exact match against weaknesses array
  • limit (number) limit results

Returns: { count: number, pokemon: Pokemon[] }

Example: /api/pokemon?type=Fire&limit=5

GET /api/pokemonById

Methods: GET, HEAD

Query Params (required):

  • id (number) – required

Returns: Pokemon object or { error }

Example: /api/pokemonById?id=1

GET /api/pokemonByNum

Methods: GET, HEAD

Query Params (required):

  • num (string) required

Returns: Pokemon object or { error }

Example: /api/pokemonByNum?num=001

GET /api/types

Methods: GET, HEAD

Returns: { types: string[] }

Example: /api/types

POST /api/addPokemon

Methods: POST

Body Params (required): id (number), num (string), name (string), type (string)

Content-Types: application/json, application/x-www-form-urlencoded

Returns: 201 + { message, pokemon }

Example (urlencoded): id=999&num=999&name=Testmon&type=Fire,Dragon

POST /api/editPokemon

Methods: POST

Body Params: id (required), name/type/weaknesses (optional)

Content-Types: application/json, application/x-www-form-urlencoded

Returns: 204 No Content (success) or JSON error

Example (urlencoded): id=999&name=Editedmon
